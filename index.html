    Frontend Web Developer Development Application

    Chennai | Tamilnadu | Internship

    Apply now

    Job Description

    We are looking for programmers with a keen eye for design for the position of front end developer. Front end developers are responsible for ensuring the alignment of web design and user experience requirements, optimizing web pages for maximum efficiency, and maintaining brand consistency across all web pages, among other duties. Front end developers are required to work in teams alongside back end developers, graphic designers, and user experience designers to ensure all elements of web creation are consistent. This requires excellent communication and interpersonal skills. You will work with the UI/UX designer and bridge the gap between graphical design and technical implementation, taking an active role on both sides and defining how the application looks as well as how it works.

    Job Summary

    Published on : Dec 5, 2021
    Vacancy : 10
    Employment Status : Internship
    Experience : 0-6 months
    Job Location: Chennai

    Responsibilites

    Build reusable code and libraries for future use.

    Designing,developing and testing UI for mobile and web applications.

    Accurately translate user and bussiness needs into functional frontend code.

    Continously maintaining the template engine.

    Maintain and improve website and also optimize applications for maximum speed.

    Requirements

    Bachelor's degree or equivalent in Computer Science.

    Any undergraduate can apply.

    Proven work experience as a Front-end developer (personal projects are also considered).

    Proficient understanding of web markup languages,including HTML5,CSS3.

    Familiarity with code versioning tools such as Git etc..,

    Perks

    Internship Certificate with Global link.

    Letter of Recommendation (Based on work ).

    Flexible Timings
